# Employee Payroll Management System (EPMS)
## ST KIZITO SAVE TVET SCHOOL - National Practical Exam 2025

### Project Structure
- backend-project/ - Node.js + Express.js backend
- frontend-project/ - React.js frontend with Tailwind CSS

### Database Schema
- EPMS Database with Employee, Department, and Salary tables
- Initial department data pre-populated

### Features
- User authentication (login/logout)
- CRUD operations for Employee, Department, and Salary
- Monthly payroll report generation
- Responsive web design

### Technologies Used
- Backend: Node.js, Express.js, MySQL/MongoDB
- Frontend: React.js, Tailwind CSS, Axios
- Authentication: Session-based login system
